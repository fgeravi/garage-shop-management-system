# Garage Shop Management System

A Java console application designed to simulate a performance garage scheduling and management system.

This project allows users to:
- Create customer appointments
- Assign technicians based on vehicle specialty
- Track appointment statuses
- View available scheduling times
- Search customers
- Calculate estimated service costs
- Track completed job revenue

The system was built using object-oriented programming principles in Java.

## Features

- Technician specialty matching
- Appointment scheduling
- Status updates
- Revenue tracking
- Interactive menu system
- Customer searching
- Dynamic appointment IDs

## Technologies

- Java
- Object-Oriented Programming
- ArrayLists
- Enums
- Console-based UI
